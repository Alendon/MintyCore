BeginSecondaryCommandBuffer(CommandBufferUsageFlags, ManagedRenderPass, ManagedFrameBuffer, UInt32, Boolean, QueryControlFlags, QueryPipelineStatisticFlags)

Begins recording a secondary command buffer.

Declaration
public void BeginSecondaryCommandBuffer(CommandBufferUsageFlags flags, ManagedRenderPass renderPass, ManagedFrameBuffer frameBuffer = null, uint subpass = 0U, bool occlusionQuery = false, QueryControlFlags queryFlags = QueryControlFlags.None, QueryPipelineStatisticFlags pipelineStatistics = QueryPipelineStatisticFlags.None)
Parameters
Type Name Description
Silk.NET.Vulkan.CommandBufferUsageFlags flags

Usage flags for the command buffer.
ManagedRenderPass renderPass

The render pass associated with the command buffer
ManagedFrameBuffer frameBuffer

The frame buffer associated with the command buffer (optional).
System.UInt32 subpass

The subpass index within the render pass (default is 0).
System.Boolean occlusionQuery

Specifies if occlusion queries are enabled (default is false).
Silk.NET.Vulkan.QueryControlFlags queryFlags

Flags controlling the behavior of occlusion queries (default is 0).
Silk.NET.Vulkan.QueryPipelineStatisticFlags pipelineStatistics

Flags controlling the statistics that are accumulated for the pipeline (default is 0).
Remarks

The render pass defines the compatible render passes when executing the secondary command buffer. It does not have to be the exact same render pass. The frame buffer must be compatible with the render pass. Passing the exact frame buffer may result in better performance.
